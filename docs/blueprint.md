# **App Name**: Dayflow HR

## Core Features:

- Secure Authentication: Implement secure sign-up and sign-in functionality using email and password authentication. Enforce password security rules and input validation. Authentication and authorization are handled through custom backend logic with role-based access control for Admin/HR and Employee users.
- Employee Profile Management: Employees can view their personal and job-related information and edit limited fields such as contact details and profile picture. Admins and HR officers can manage all employee details, including role assignments and employment information.
- Attendance Tracking: Provide daily and weekly attendance views with check-in and check-out functionality. Attendance records support multiple status types, including Present, Absent, Half-day, and Leave, and are stored and retrieved dynamically from a relational database.
- Leave Management: Allow employees to apply for different types of leave, such as Paid, Sick, and Unpaid, with remarks and date ranges. Admins and HR officers can review, approve, or reject leave requests and add comments. All updates are reflected immediately in employee records.
- Payroll Visibility: Employees have read-only access to their payroll information. Admins and HR officers can view and update salary structures while ensuring data accuracy and consistency.

## Style Guidelines:

- Use a clean and intuitive layout with a clear visual hierarchy to support easy navigation and efficient task completion.
- Apply Inter for body text and Space Grotesk for headings to maintain readability while preserving a modern, professional appearance.
- Use consistent and professional icons to represent HR functions and improve usability through clear visual cues.
- Incorporate subtle animations and transitions only where necessary to provide user feedback, without distracting from functionality.
